General info + both amenity forms — matched to your real types

Rebuilt against the actual AmenityTypeFormProps and Input types you shared, rather than the shape I'd been assuming. Two things changed as a direct result:

  • AmenityTypeFormProps has no generalInfo/amenity object — just amenityTypeId: number, amenityId, isEditing, locationFunctionId, locationSlug, onClose. All three components below now destructure exactly these fields, nothing invented. A nice side effect: since amenityTypeId is a real number here (not the number | '' the shell uses internally), there's no "please select a type" check needed inside these forms anymore — the shell already guarantees a valid type before mounting them.
  • Input doesn't expose onBlur. RHF's register() spread returns onBlur among its props, which would fail to type-check against Input's declared prop list, and mode: 'onBlur' validation wouldn't have anywhere to fire anyway. Switched GeneralInfoFields to mode: 'onChange' and wired the name field through Controller instead of register, passing error straight into Input's own built-in error prop rather than rendering a separate error paragraph for that field.
Please verify: useInsertAmenity / useUpdateAmenity are called below as (locationFunctionId, locationSlug) and (amenityId, locationSlug) respectively, based on AmenityTypeFormProps carrying both fields. Your actual hook signatures have appeared slightly differently across earlier screenshots (sometimes a single argument) — double check against the current hook files before wiring this in, since a mismatched argument here would silently pass the wrong value where the hook expects something else.
components/GeneralInfoFields.tsx changed

Name field now goes through Controller + your real Input component, using its native error/label props instead of a hand-rolled error paragraph. Description stays a plain textarea since no Storybook textarea equivalent was shown.

'use client';

import { forwardRef, useImperativeHandle } from 'react';
import { useForm, Controller } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { Input } from '@hlt/react-component-library';
import { generalInfoSchema, type GeneralInfoFormValues } from '../lib/general-info-schema';

export interface GeneralInfoFieldsHandle {
  validate: () => Promise<boolean>;
  getValues: () => GeneralInfoFormValues;
  reset: (values: GeneralInfoFormValues) => void;
}

interface Props {
  defaultValues?: Partial<GeneralInfoFormValues>;
}

export const GeneralInfoFields = forwardRef<GeneralInfoFieldsHandle, Props>(
  function GeneralInfoFields({ defaultValues }, ref) {
    const {
      control,
      trigger,
      getValues,
      reset,
      formState: { errors },
    } = useForm<GeneralInfoFormValues>({
      resolver: zodResolver(generalInfoSchema),
      defaultValues: { amenityName: '', amenityDescription: '', ...defaultValues },
      // Input doesn't expose onBlur, so 'onBlur' mode has nothing to trigger
      // off of -- validate as the user types instead.
      mode: 'onChange',
    });

    useImperativeHandle(ref, () => ({ validate: () => trigger(), getValues, reset }));

    return (
      <div className="flex flex-col gap-4">
        <Controller
          name="amenityName"
          control={control}
          render={({ field }) => (
            <Input
              ref={field.ref}
              name={field.name}
              value={field.value}
              onChange={field.onChange}
              label="Name"
              placeholder="e.g. General Admission"
              className=" bg-white"
              error={errors.amenityName?.message}
            />
          )}
        />

        <div className="flex flex-col gap-1">
          <label className="text-sm font-medium text-gray-700">Description</label>
          <Controller
            name="amenityDescription"
            control={control}
            render={({ field }) => (
              <textarea
                rows={3}
                className="w-full rounded-md border border-gray-300 bg-white px-3 py-2 text-sm text-gray-900 focus:outline-none focus:ring-2 focus:ring-blue-500"
                placeholder="Short description (optional)"
                {...field}
              />
            )}
          />
          {errors.amenityDescription && (
            <p className="text-xs text-red-600 mt-1">{errors.amenityDescription.message}</p>
          )}
        </div>
      </div>
    );
  },
);

If Input's internal implementation does actually spread an onBlur through to the native <input> (just not declared in its public type), you could switch back to mode: 'onBlur' and pass onBlur={field.onBlur} alongside the other Controller props with an as any/type augmentation. Safer to leave as onChange mode unless you've confirmed that from the component's source.

components/forms/GeneralAdmissionForm.tsx changed

Destructures exactly the fields AmenityTypeFormProps declares -- no invented generalInfo/amenity object. Since amenityTypeId is guaranteed a valid number by the time this mounts, the old "please select a type" check is gone.

'use client';

import { useEffect, useRef } from 'react';
import { useInsertAmenity } from '../../hooks/useInsertAmenity';
import { useUpdateAmenity } from '../../hooks/useUpdateAmenity';
import { useAmenityGeneralInfo } from '../../hooks/useAmenityGeneralInfo';
import { GeneralInfoFields, type GeneralInfoFieldsHandle } from '../GeneralInfoFields';
import type { AmenityTypeFormProps } from '@/types/amenities';
import { Button } from '@hlt/react-component-library';

export function GeneralAdmissionForm({
  amenityTypeId,
  isEditing,
  amenityId,
  locationFunctionId,
  locationSlug,
  onClose,
}: AmenityTypeFormProps) {
  const formRef = useRef<GeneralInfoFieldsHandle>(null);
  const hasHydrated = useRef(false);

  const { mutate: insertMutate, isPending: insertIsPending } =
    useInsertAmenity(locationFunctionId, locationSlug);
  const { mutate: updateAmenityGeneralInfoMutate, isPending: updateGeneralInfoPending } =
    useUpdateAmenity(amenityId ?? 0, locationSlug);

  const isSaving = insertIsPending || updateGeneralInfoPending;

  const {
    data: fetchedGeneralInfo,
    isLoading: isLoadingGeneral,
    isError: isGeneralError,
    error: generalError,
  } = useAmenityGeneralInfo(isEditing ? (amenityId ?? null) : null);

  useEffect(() => {
    if (fetchedGeneralInfo && !hasHydrated.current) {
      formRef.current?.reset({
        amenityName: fetchedGeneralInfo.amenityName,
        amenityDescription: fetchedGeneralInfo.amenityDescription ?? '',
      });
      hasHydrated.current = true;
    }
  }, [fetchedGeneralInfo]);

  async function handleSave() {
    const isValid = await formRef.current?.validate();
    if (!isValid) return;

    const values = formRef.current!.getValues();
    const payload = {
      amenityTypeId,
      amenityName: values.amenityName,
      amenityDescription: values.amenityDescription ?? '',
    };

    if (isEditing && amenityId) {
      updateAmenityGeneralInfoMutate(
        { type: 'client', amenityId, ...payload },
        { onSuccess: (data) => { if (data.updateAmenity.success) onClose(); } },
      );
    } else {
      insertMutate(
        { type: 'client', locationFunctionId, ...payload },
        { onSuccess: (data) => { if (data.insertAmenity.success) onClose(); } },
      );
    }
  }

  const isLoadingInitial = isEditing && isLoadingGeneral && !hasHydrated.current;

  if (isLoadingInitial) {
    return (
      <div className="flex flex-col gap-3 animate-pulse">
        <div className="h-9 bg-gray-100 rounded-md" />
        <div className="h-24 bg-gray-100 rounded-md" />
      </div>
    );
  }

  if (isEditing && isGeneralError) {
    return <p className="text-sm text-red-600">Could not load this amenity. {generalError?.message}</p>;
  }

  return (
    <div className="flex flex-col gap-4">
      <GeneralInfoFields ref={formRef} />
      <div className="flex justify-end gap-2 pt-3 border-t border-gray-100 mt-1">
        <Button
          className="text-center px-4 w-40 text-white"
          onClick={onClose}
          disabled={isSaving}
          loading={isSaving}
          size="md"
          variant="buttonGrey"
        >
          Cancel
        </Button>
        <Button
          className="text-center px-4 w-40"
          onClick={handleSave}
          disabled={isSaving}
          loading={isSaving}
          size="md"
        >
          {isSaving ? 'Saving...' : 'Save'}
        </Button>
      </div>
    </div>
  );
}
components/forms/ScheduledAdmissionForm.tsx changed

Same prop-shape fix applied. amenityId starts from the prop (for edit mode) and is promoted to local state once an insert succeeds (for create mode), exactly as in your current version -- this part didn't need to change, since it already matched how the real data flows.

'use client';

import { useEffect, useRef, useState } from 'react';
import { useInsertAmenity } from '../../hooks/useInsertAmenity';
import { useUpdateAmenity } from '../../hooks/useUpdateAmenity';
import { useUpsertAmenitySchedule } from '../../hooks/useUpsertAmenitySchedule';
import { useAmenityGeneralInfo } from '../../hooks/useAmenityGeneralInfo';
import { useAmenitySchedule } from '../../hooks/useAmenitySchedule';
import { GeneralInfoFields, type GeneralInfoFieldsHandle } from '../GeneralInfoFields';
import { RecurrenceRuleBuilder, type RecurrenceRuleBuilderHandle } from '../schedule/RecurrenceRuleBuilder';
import { buildScheduleSavePayload } from '../../lib/schedule-recurrence-engine';
import type { AmenityTypeFormProps } from '@/types/amenities';
import { Button, Tabs } from '@hlt/react-component-library';

export function ScheduledAdmissionForm({
  amenityTypeId,
  isEditing,
  amenityId: amenityIdProp,
  locationFunctionId,
  locationSlug,
  onClose,
}: AmenityTypeFormProps) {
  const [activeTab, setActiveTab] = useState('general');
  const [amenityId, setAmenityId] = useState<number | null>(amenityIdProp ?? null);

  const generalFormRef = useRef<GeneralInfoFieldsHandle>(null);
  const scheduleFormRef = useRef<RecurrenceRuleBuilderHandle>(null);
  const hasHydratedGeneral = useRef(false);
  const hasHydratedSchedule = useRef(false);

  const {
    mutate: insertMutate,
    isPending: insertIsPending,
    data: insertSuccessData,
  } = useInsertAmenity(locationFunctionId, locationSlug);
  const { mutate: updateAmenityGeneralInfoMutate, isPending: updateGeneralInfoPending } =
    useUpdateAmenity(amenityId ?? 0, locationSlug);
  const { mutate: upsertAmenitySchedule, isPending: updateScheduleInfoPending } =
    useUpsertAmenitySchedule(amenityId ?? 0);

  const isSaving = insertIsPending || updateGeneralInfoPending || updateScheduleInfoPending;
  const amenityIdFromInsert = insertSuccessData?.insertAmenity?.data?.amenityId ?? null;

  const {
    data: fetchedGeneralInfo,
    isLoading: isLoadingGeneral,
    isError: isGeneralError,
    error: generalError,
  } = useAmenityGeneralInfo(isEditing ? (amenityId ?? null) : null);

  const {
    data: fetchedSchedule,
    isLoading: isLoadingSchedule,
    isError: isScheduleError,
    error: scheduleError,
  } = useAmenitySchedule(
    isEditing ? (amenityId ?? null) : null,
    activeTab === 'scheduler',
  );

  useEffect(() => {
    if (amenityIdFromInsert) setAmenityId(amenityIdFromInsert);
  }, [amenityIdFromInsert]);

  useEffect(() => {
    if (fetchedGeneralInfo && !hasHydratedGeneral.current) {
      generalFormRef.current?.reset({
        amenityName: fetchedGeneralInfo.amenityName,
        amenityDescription: fetchedGeneralInfo.amenityDescription ?? '',
      });
      hasHydratedGeneral.current = true;
    }
  }, [fetchedGeneralInfo]);

  useEffect(() => {
    if (fetchedSchedule && fetchedSchedule.length && !hasHydratedSchedule.current) {
      const { scheduleJson } = fetchedSchedule[0];
      const scheduleJsonParsed = JSON.parse(scheduleJson).rule;
      scheduleFormRef.current?.reset(scheduleJsonParsed);
      hasHydratedSchedule.current = true;
    }
  }, [fetchedSchedule]);

  async function handleNext() {
    const isValid = await generalFormRef.current?.validate();
    if (!isValid) return;

    const values = generalFormRef.current!.getValues();
    const payload = {
      amenityTypeId,
      amenityName: values.amenityName,
      amenityDescription: values.amenityDescription ?? '',
    };

    if (isEditing && amenityId) {
      updateAmenityGeneralInfoMutate(
        { type: 'client', amenityId, ...payload },
        { onSuccess: (data) => { if (data.updateAmenity.success) setActiveTab('scheduler'); } },
      );
    } else {
      insertMutate(
        { type: 'client', locationFunctionId, ...payload },
        { onSuccess: (data) => { if (data.insertAmenity.success) setActiveTab('scheduler'); } },
      );
    }
  }

  async function handleSave() {
    const isValid = await scheduleFormRef.current?.validate();
    if (!isValid) return;

    const recurrenceRule = scheduleFormRef.current!.getValues();
    const { occurrences, rule, rruleText } = buildScheduleSavePayload(recurrenceRule);
    const payload = {
      amenityId: amenityId || 0,
      scheduleJson: JSON.stringify({ rule, rruleText }),
      datesJson: JSON.stringify(occurrences),
    };
    upsertAmenitySchedule(
      { type: 'client', ...payload },
      { onSuccess: (data) => { if (data.upsertAmenitySchedule.success) onClose(); } },
    );
  }

  const isLoadingInitialGeneral = isEditing && isLoadingGeneral && !hasHydratedGeneral.current;
  const isLoadingInitialSchedule = isEditing && isLoadingSchedule && !hasHydratedSchedule.current;

  const tabs = [
    {
      id: 'general',
      label: 'General',
      content: isLoadingInitialGeneral ? (
        <div className="flex flex-col gap-3 animate-pulse">
          <div className="h-9 bg-gray-100 rounded-md" />
          <div className="h-24 bg-gray-100 rounded-md" />
        </div>
      ) : isGeneralError ? (
        <p className="text-sm text-red-600">
          Could not load general info. {generalError?.message}
        </p>
      ) : (
        <>
          <GeneralInfoFields ref={generalFormRef} />
          <div className="flex justify-end gap-2 pt-3 border-t border-gray-100 mt-1">
            <Button
              className="text-center px-4 w-40 text-white"
              onClick={onClose}
              disabled={isSaving}
              loading={isSaving}
              size="md"
              variant="buttonGrey"
            >
              Cancel
            </Button>
            <Button
              className="text-center px-4 w-40"
              onClick={handleNext}
              disabled={isSaving}
              loading={isSaving}
              size="md"
            >
              Next
            </Button>
          </div>
        </>
      ),
    },
    {
      id: 'scheduler',
      label: 'Scheduler',
      disabled: !amenityId,
      content: isLoadingInitialSchedule ? (
        <div className="flex flex-col gap-3 animate-pulse">
          <div className="h-9 bg-gray-100 rounded-md" />
          <div className="h-9 bg-gray-100 rounded-md" />
          <div className="h-32 bg-gray-100 rounded-md" />
        </div>
      ) : isScheduleError ? (
        <p className="text-sm text-red-600">
          Could not load the schedule. {scheduleError?.message}
        </p>
      ) : (
        <>
          <RecurrenceRuleBuilder ref={scheduleFormRef} />
          <div className="flex justify-between gap-2 pt-3 border-t border-gray-100 mt-6">
            <Button
              className="text-center px-4 w-38"
              onClick={() => setActiveTab('general')}
              disabled={isSaving}
              loading={isSaving}
              size="md"
              variant="outlineAccent"
            >
              Back
            </Button>
            <div className="flex gap-2">
              <Button
                className="text-center px-4 w-40 text-white"
                onClick={onClose}
                disabled={isSaving}
                loading={isSaving}
                size="md"
                variant="buttonGrey"
              >
                Cancel
              </Button>
              <Button
                className="text-center px-4 w-40"
                onClick={handleSave}
                disabled={isSaving}
                loading={isSaving}
                size="md"
              >
                {isSaving ? 'Saving...' : 'Save'}
              </Button>
            </div>
          </div>
        </>
      ),
    },
  ];

  return (
    <div className="flex flex-col gap-4">
      <Tabs tabs={tabs} activeTabId={activeTab} onTabChange={(id) => setActiveTab(id)} />
    </div>
  );
}
