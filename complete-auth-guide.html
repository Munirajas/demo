Q1 — graphql-request + TanStack Query is fine for your app
Stay with graphql-request + TanStack Query. For your current situation it is the right call. Apollo setup has been painful and the benefit (normalized cache) is only worth it when you have complex cross-component data relationships. You can always migrate later. The manual cache invalidation with invalidateQueries is straightforward once you see the pattern.

Q3 — Session refresh flows
FLOW A: Active user — token expires mid-session useAuthQuery fires API call ↓ gets 401 / UNAUTHENTICATED POST /api/auth/refresh ↓ refreshAuth(userId, token) on BE new tokens → cookies updated original query retried automatically user sees nothing — transparent FLOW B: Idle user returns to app Tab becomes visible → visibilitychange fires ↓ TokenRefreshProvider checks token exp if expiring in < 5min → proactive refresh if refresh ok → continues normally if refresh fails → redirect /login?reason=expired FLOW C: Proactive (every 4 minutes) TokenRefreshProvider interval fires decodes JWT exp claim if < 5min remaining → refresh now user never hits 401 at all
Q4 — Logout needs both BE call AND cookie clear
WHY you must call the BE on logout: If you ONLY clear cookies, the tokens still work until they naturally expire (could be 24h) For enterprise SaaS this is a security risk Your schema has endSession mutation → use it LOGOUT FLOW (logoutAction.ts): User clicks LogoutButton ↓ queryClient.clear() ← clears TanStack cache FIRST ↓ logoutAction() Server Action 1. endSession mutation → BE invalidates tokens 2. clearAuthCookies() → all 4 cookies deleted 3. redirect('/login') Even if endSession fails → cookies are ALWAYS cleared
Q5 — User details and including cookies in further calls
USER DETAILS (zero extra API calls): idToken is already stored as a cookie after login It contains: email, name, sub (userId), etc. GET /api/auth/session decodes it server-side useSession() hook calls this → returns SessionUser name, email, initials available instantly HOW COOKIES ARE INCLUDED IN FURTHER API CALLS: Server Components: gqlFetch() reads access_token cookie directly sets Authorization: Bearer token header you never touch cookies manually Client Components: useAuthQuery() fetches GET /api/auth/token /api/auth/token reads httpOnly cookie server-side returns token → hook sets Authorization header you never touch cookies manually

SERVER
All server-side GQL calls go through gqlFetch(). Auto-refreshes token. Use this in Server Components.
import 'server-only';

// ─────────────────────────────────────────────────────────────
// GQL CLIENT — SERVER ONLY
// Single factory for all server-side GraphQL calls.
// Automatically attaches Bearer token from cookies.
// ─────────────────────────────────────────────────────────────

import { GraphQLClient, ClientError } from 'graphql-request';
import { getAccessToken } from '@/modules/auth/lib/cookies';
import { refreshAccessToken } from '@/modules/auth/lib/refresh';

const GQL_ENDPOINT = process.env.NEXT_PUBLIC_GRAPHQL_URL!;

// ── Build a client with the given token ───────────────────────
function createClient(token?: string): GraphQLClient {
  return new GraphQLClient(GQL_ENDPOINT, {
    headers: {
      'Content-Type': 'application/json',
      ...(token && { Authorization: `Bearer ${token}` }),
    },
  });
}

// ── Public client — no token (for login mutation) ────────────
export function createPublicClient(): GraphQLClient {
  return createClient();
}

// ── Authenticated client — reads token from cookie ───────────
// Auto-refreshes if token is expired.
// Throws 'SESSION_EXPIRED' if refresh also fails.
export async function createAuthenticatedClient(): Promise<GraphQLClient> {
  let token = await getAccessToken();

  if (!token) {
    // Try silent refresh before giving up
    token = await refreshAccessToken();
    if (!token) throw new Error('SESSION_EXPIRED');
  }

  return createClient(token);
}

// ── Helper — typed GQL fetch with auto-refresh ───────────────
export async function gqlFetch<
  TData,
  TVars extends object = Record<string, never>
>(
  query: string,
  variables?: TVars
): Promise<TData> {
  const client = await createAuthenticatedClient();

  try {
    return await client.request<TData>(query, variables);
  } catch (error) {
    // Token expired mid-request — refresh and retry once
    if (isAuthError(error)) {
      const newToken = await refreshAccessToken();
      if (!newToken) throw new Error('SESSION_EXPIRED');
      return createClient(newToken).request<TData>(query, variables);
    }
    throw error;
  }
}

function isAuthError(error: unknown): boolean {
  if (error instanceof ClientError) {
    const status = error.response?.status;
    const code   = error.response?.errors?.[0]?.extensions?.code as string | undefined;
    return status === 401 || code === 'UNAUTHENTICATED';
  }
  return false;
}

export { GQL_ENDPOINT };

SERVER
Added getSessionUser() which decodes idToken for user profile. No extra API call.
import 'server-only';

import { cookies } from 'next/headers';
import type { AuthToken, DecodedIdToken, SessionUser } from '@/types/auth.types';

const COOKIE_KEYS = {
  ACCESS_TOKEN:  'access_token',
  ID_TOKEN:      'id_token',
  REFRESH_TOKEN: 'refresh_token',
  USER_ID:       'user_id',
} as const;

const BASE_OPTIONS = {
  httpOnly: true,
  secure:   process.env.NODE_ENV === 'production',
  sameSite: 'lax' as const,
  path:     '/',
};

const ACCESS_OPTIONS  = { ...BASE_OPTIONS, maxAge: 60 * 60 * 24 };       // 24h
const REFRESH_OPTIONS = { ...BASE_OPTIONS, maxAge: 60 * 60 * 24 * 30 };  // 30d

// ── Decode JWT without verification ──────────────────────────
export function decodeToken<T>(token: string): T | null {
  try {
    const base64 = token.split('.')[1]
      .replace(/-/g, '+')
      .replace(/_/g, '/');
    return JSON.parse(Buffer.from(base64, 'base64').toString('utf-8')) as T;
  } catch {
    return null;
  }
}

// ── WRITE — after login or refresh ───────────────────────────
export async function setAuthCookies(tokens: AuthToken): Promise<void> {
  const store = await cookies();
  store.set(COOKIE_KEYS.ACCESS_TOKEN,  tokens.token,         ACCESS_OPTIONS);
  store.set(COOKIE_KEYS.ID_TOKEN,      tokens.idToken,       ACCESS_OPTIONS);
  store.set(COOKIE_KEYS.REFRESH_TOKEN, tokens.refreshToken,  REFRESH_OPTIONS);
  store.set(COOKIE_KEYS.USER_ID,       tokens.userId,        ACCESS_OPTIONS);
}

// ── READ ──────────────────────────────────────────────────────
export async function getAccessToken()  { return (await cookies()).get(COOKIE_KEYS.ACCESS_TOKEN)?.value;  }
export async function getRefreshToken() { return (await cookies()).get(COOKIE_KEYS.REFRESH_TOKEN)?.value; }
export async function getIdToken()      { return (await cookies()).get(COOKIE_KEYS.ID_TOKEN)?.value;      }
export async function getUserId()       { return (await cookies()).get(COOKIE_KEYS.USER_ID)?.value;       }

// ── Get session user from stored cookies ─────────────────────
// Used by Server Components to pass user info to client tree
export async function getSessionUser(): Promise<SessionUser | null> {
  const [idToken, userId] = await Promise.all([getIdToken(), getUserId()]);
  if (!idToken || !userId) return null;

  const payload = decodeToken<DecodedIdToken>(idToken);
  if (!payload) return null;

  const email = payload.email ?? '';
  const name = payload.name
    ?? payload['cognito:username']
    ?? `${payload.given_name ?? ''} ${payload.family_name ?? ''}`.trim()
    ?? email.split('@')[0];

  // Build initials from name
  const parts = name.trim().split(' ').filter(Boolean);
  const initials = parts.length >= 2
    ? `${parts[0][0]}${parts[parts.length - 1][0]}`.toUpperCase()
    : name.slice(0, 2).toUpperCase();

  return { userId, email, name, initials };
}

// ── Check if access token is close to expiry ─────────────────
// Used for proactive refresh (refresh before user hits 401)
export async function isTokenExpiringSoon(thresholdMinutes = 5): Promise<boolean> {
  const token = await getAccessToken();
  if (!token) return true;

  const payload = decodeToken<DecodedIdToken>(token);
  if (!payload?.exp) return true;

  const expiresInMs = (payload.exp * 1000) - Date.now();
  return expiresInMs < thresholdMinutes * 60 * 1000;
}

// ── CLEAR — on logout ─────────────────────────────────────────
export async function clearAuthCookies(): Promise<void> {
  const store = await cookies();
  Object.values(COOKIE_KEYS).forEach((key) => store.delete(key));
}
refresh.ts
src/modules/auth/lib/refresh.ts
SERVER
Calls refreshAuth(userId, token) — matches your exact schema.
import 'server-only';

import { GraphQLClient, ClientError } from 'graphql-request';
import { GQL_ENDPOINT } from '@/lib/gql-client';
import { getAccessToken, getUserId, setAuthCookies, clearAuthCookies } from '@/modules/auth/lib/cookies';
import type { RefreshAuthMutationResponse } from '@/types/auth.types';

const REFRESH_AUTH_MUTATION = `
  mutation RefreshAuth($userId: String!, $token: String!) {
    refreshAuth(userId: $userId, token: $token) {
      token
      idToken
      refreshToken
      userId
    }
  }
`;

export async function refreshAccessToken(): Promise<string | null> {
  const [token, userId] = await Promise.all([getAccessToken(), getUserId()]);

  if (!token || !userId) {
    console.warn('[refresh] Missing token or userId');
    return null;
  }

  try {
    const client = new GraphQLClient(GQL_ENDPOINT, {
      headers: { 'Content-Type': 'application/json' },
    });

    const data = await client.request<RefreshAuthMutationResponse>(
      REFRESH_AUTH_MUTATION,
      { userId, token }
    );

    if (!data?.refreshAuth?.token) return null;

    await setAuthCookies(data.refreshAuth);
    return data.refreshAuth.token;

  } catch (error) {
    if (error instanceof ClientError) {
      const msg = (error.response?.errors?.[0]?.message ?? '').toLowerCase();
      // If tokens are truly expired/invalid — clear everything
      if (msg.includes('expired') || msg.includes('invalid') || msg.includes('unauthenticated')) {
        await clearAuthCookies();
      }
    }
    console.error('[refresh] Failed:', error);
    return null;
  }
}

⚠ Sends userName not email — confirmed from your schema. Maps form field name to BE field name.
'use server';

import { GraphQLClient, ClientError } from 'graphql-request';
import { redirect } from 'next/navigation';
import { GQL_ENDPOINT } from '@/lib/gql-client';
import { setAuthCookies } from '@/modules/auth/lib/cookies';
import type { LoginMutationResponse, ActionResult } from '@/types/auth.types';

// ⚠️ BE uses 'userName' not 'email' — confirmed from schema screenshot
const LOGIN_MUTATION = `
  mutation Login($userName: String!, $password: String!) {
    login(userName: $userName, password: $password) {
      token
      idToken
      refreshToken
      userId
    }
  }
`;

export async function loginAction(
  input: { email: string; password: string }
): Promise<ActionResult> {
  const { email, password } = input;

  if (!email?.trim()) return { success: false, error: 'Email is required.' };
  if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email.trim()))
    return { success: false, error: 'Enter a valid email.' };
  if (!password) return { success: false, error: 'Password is required.' };

  try {
    const client = new GraphQLClient(GQL_ENDPOINT, {
      headers: { 'Content-Type': 'application/json' },
    });

    const data = await client.request<LoginMutationResponse>(LOGIN_MUTATION, {
      userName: email.trim().toLowerCase(), // maps form 'email' → BE 'userName'
      password,
    });

    if (!data?.login?.token || !data.login.userId) {
      return { success: false, error: 'Login failed: incomplete response from server.' };
    }

    await setAuthCookies(data.login);

  } catch (error) {
    if (error instanceof ClientError) {
      const msg = error.response?.errors?.[0]?.message ?? '';
      if (msg.toLowerCase().includes('invalid') || msg.toLowerCase().includes('incorrect'))
        return { success: false, error: 'Invalid email or password.' };
      if (msg.toLowerCase().includes('missing tenants'))
        return { success: false, error: 'Account not configured. Contact your administrator.' };
      if (msg.toLowerCase().includes('disabled') || msg.toLowerCase().includes('blocked'))
        return { success: false, error: 'Account disabled. Contact support.' };
      return { success: false, error: msg || 'Login failed. Please try again.' };
    }
    console.error('[loginAction]', error);
    return { success: false, error: 'Unable to connect. Check your connection.' };
  }

  redirect('/dashboard'); // outside try/catch — critical
}

'use server';

import { GraphQLClient, ClientError } from 'graphql-request';
import { redirect } from 'next/navigation';
import { GQL_ENDPOINT } from '@/lib/gql-client';
import { getAccessToken, getUserId, clearAuthCookies } from '@/modules/auth/lib/cookies';
import type { ActionResult } from '@/types/auth.types';

// Seen in your schema screenshot — endSession SessionResponse
const END_SESSION_MUTATION = `
  mutation EndSession {
    endSession {
      success
    }
  }
`;

export async function logoutAction(): Promise<ActionResult> {
  const [token, userId] = await Promise.all([getAccessToken(), getUserId()]);

  // ── Step 1: Inform BE to invalidate the session ───────────
  // This revokes the token server-side (Cognito + BE session)
  // Even if this fails, we STILL clear cookies — user must be logged out
  if (token) {
    try {
      const client = new GraphQLClient(GQL_ENDPOINT, {
        headers: {
          'Content-Type': 'application/json',
          Authorization: `Bearer ${token}`,
        },
      });
      await client.request(END_SESSION_MUTATION);
    } catch (error) {
      // Log but don't block logout — always clear cookies
      if (error instanceof ClientError) {
        console.warn('[logoutAction] endSession BE call failed:', error.response?.errors?.[0]?.message);
      } else {
        console.warn('[logoutAction] endSession network error');
      }
    }
  }

  // ── Step 2: Clear all 4 httpOnly cookies ──────────────────
  await clearAuthCookies();

  // ── Step 3: Redirect to login ─────────────────────────────
  redirect('/login');
}
useAuthQuery.ts
src/modules/auth/hooks/useAuthQuery.ts
CLIENT
Drop-in replacement for Apollo useQuery/useMutation. Auto token refresh. invalidateKeys for cache management.
'use client';

// ─────────────────────────────────────────────────────────────
// useAuthQuery — TanStack Query wrapper with auto token refresh
//
// This replaces Apollo Client's auth link for client components.
// Every query goes through this hook which:
//   1. Gets the access token from /api/auth/token
//   2. Makes the GraphQL request with Bearer token
//   3. On 401 → calls /api/auth/refresh → retries once
//   4. On failed refresh → redirects to /login
//
// Usage:
//   const { data, isLoading, error } = useAuthQuery(
//     ['users'],
//     USERS_QUERY,
//     { role: 'admin' }  // optional variables
//   );
// ─────────────────────────────────────────────────────────────

import { useQuery, useMutation, useQueryClient, type UseQueryOptions } from '@tanstack/react-query';
import { useRouter } from 'next/navigation';
import { GraphQLClient, ClientError } from 'graphql-request';

const GQL_ENDPOINT = process.env.NEXT_PUBLIC_GRAPHQL_URL!;

// ── Fetch access token from cookie bridge ────────────────────
async function getToken(): Promise<string | null> {
  try {
    const res = await fetch('/api/auth/token');
    if (!res.ok) return null;
    const { accessToken } = await res.json() as { accessToken: string | null };
    return accessToken;
  } catch {
    return null;
  }
}

// ── Try to refresh token ──────────────────────────────────────
async function tryRefresh(): Promise<string | null> {
  try {
    const res = await fetch('/api/auth/refresh', { method: 'POST' });
    if (!res.ok) return null;
    const { accessToken } = await res.json() as { accessToken: string | null };
    return accessToken;
  } catch {
    return null;
  }
}

// ── Core GQL fetcher — used by both useAuthQuery and useAuthMutation
async function gqlFetch<TData>(
  query: string,
  variables?: object,
  onSessionExpired?: () => void
): Promise<TData> {
  let token = await getToken();

  if (!token) {
    token = await tryRefresh();
    if (!token) { onSessionExpired?.(); throw new Error('SESSION_EXPIRED'); }
  }

  const client = new GraphQLClient(GQL_ENDPOINT, {
    headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json' },
  });

  try {
    return await client.request<TData>(query, variables);
  } catch (error) {
    if (isAuthError(error)) {
      // Token expired mid-request — refresh once and retry
      const newToken = await tryRefresh();
      if (!newToken) { onSessionExpired?.(); throw new Error('SESSION_EXPIRED'); }

      const retryClient = new GraphQLClient(GQL_ENDPOINT, {
        headers: { Authorization: `Bearer ${newToken}`, 'Content-Type': 'application/json' },
      });
      return retryClient.request<TData>(query, variables);
    }
    throw error;
  }
}

function isAuthError(error: unknown): boolean {
  if (error instanceof ClientError) {
    const code = error.response?.errors?.[0]?.extensions?.code as string | undefined;
    return error.response?.status === 401 || code === 'UNAUTHENTICATED';
  }
  return false;
}

// ── useAuthQuery — for data fetching ─────────────────────────
export function useAuthQuery<TData>(
  queryKey: unknown[],
  query: string,
  variables?: object,
  options?: Omit<UseQueryOptions<TData>, 'queryKey' | 'queryFn'>
) {
  const router = useRouter();

  return useQuery<TData>({
    queryKey,
    queryFn: () =>
      gqlFetch<TData>(query, variables, () => router.push('/login?reason=expired')),
    staleTime: 1000 * 60 * 5,       // 5 min — don't refetch if data is fresh
    retry: 1,                        // retry once on network error
    ...options,
  });
}

// ── useAuthMutation — for mutations ───────────────────────────
export function useAuthMutation<TData, TVars extends object = object>(
  mutation: string,
  options?: {
    onSuccess?: (data: TData) => void;
    onError?: (error: Error) => void;
    invalidateKeys?: unknown[][];    // query keys to invalidate after mutation
  }
) {
  const router = useRouter();
  const queryClient = useQueryClient();

  return useMutation<TData, Error, TVars>({
    mutationFn: (variables: TVars) =>
      gqlFetch<TData>(mutation, variables, () => router.push('/login?reason=expired')),
    onSuccess: (data) => {
      // Invalidate specified query caches after mutation
      if (options?.invalidateKeys) {
        options.invalidateKeys.forEach((key) => {
          queryClient.invalidateQueries({ queryKey: key });
        });
      }
      options?.onSuccess?.(data);
    },
    onError: options?.onError,
  });
}

// ─────────────────────────────────────────────────────────────
// useSession — get current user in any client component
//
// Reads the idToken from /api/auth/token and decodes the

// No extra BE call needed — data is already in the token.

import { useQuery } from '@tanstack/react-query';
import type { SessionUser } from '@/types/auth.types';

async function fetchSession(): Promise<SessionUser | null> {
  const res = await fetch('/api/auth/session');
  if (!res.ok) return null;
  return res.json() as Promise<SessionUser | null>;
}

export function useSession() {
  const { data: user, isLoading } = useQuery<SessionUser | null>({
    queryKey:  ['session'],
    queryFn:   fetchSession,
    staleTime: 1000 * 60 * 10,  // 10 min — user details don't change often
    retry:     false,
  });

  return { user: user ?? null, isLoading };
}

'use client';

// ─────────────────────────────────────────────────────────────
// LOGOUT BUTTON
//
// Flow on click:
//   1. logoutAction (Server Action):
//      a. Calls endSession mutation on BE (invalidates server-side)
//      b. Clears all 4 httpOnly cookies
//      c. redirect('/login')
//   2. TanStack Query cache cleared (stale data won't leak)
//
// The queryClient.clear() is called BEFORE the server action
// so there's no moment where stale data is visible.
// ─────────────────────────────────────────────────────────────

import { useState, useTransition } from 'react';
import { useQueryClient } from '@tanstack/react-query';
import { logoutAction } from '@/modules/auth/actions/logoutAction';

interface LogoutButtonProps {
  className?: string;
  children?: React.ReactNode;
}

export function LogoutButton({ className, children }: LogoutButtonProps) {
  const queryClient = useQueryClient();
  const [isPending, startTransition] = useTransition();
  const [error, setError] = useState<string | null>(null);

  function handleLogout() {
    // Clear all cached query data immediately
    // Prevents stale data showing if another user logs in
    queryClient.clear();

    startTransition(async () => {
      const result = await logoutAction();
      // If logoutAction redirected, this never runs
      // Only runs if there was an error
      if (!result.success) {
        setError('Logout failed. Please try again.');
      }
    });
  }

  return (
    <div>
      {error && (
        <p className="text-xs text-red-400 mb-1">{error}</p>
      )}
      <button
        onClick={handleLogout}
        disabled={isPending}
        className={className ?? `
          flex items-center gap-2 w-full px-3 py-2 rounded-lg text-sm
          text-gray-400 hover:text-white hover:bg-gray-800
          transition-colors disabled:opacity-50 disabled:cursor-not-allowed
        `}
      >
        {isPending ? (
          <>
            <svg className="animate-spin w-4 h-4" viewBox="0 0 24 24" fill="none">
              <circle className="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" strokeWidth="4"/>
              <path className="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8V0C5.4 0 0 5.4 0 12h4z"/>
            </svg>
            Signing out...
          </>
        ) : (
          children ?? (

'use client';

// ─────────────────────────────────────────────────────────────
// TOKEN REFRESH PROVIDER
//
// Handles the idle user scenario:
//   - Checks token expiry every 4 minutes
//   - If token expires in < 5 minutes → refreshes proactively
//   - User never hits a 401 — completely transparent
//   - If app window becomes active after being hidden (user returns
//     from idle) → immediately checks and refreshes if needed
//
// Mount this in your (protected)/layout.tsx
// ─────────────────────────────────────────────────────────────

import { useEffect, useCallback } from 'react';
import { useQueryClient } from '@tanstack/react-query';
import { useRouter } from 'next/navigation';

const CHECK_INTERVAL_MS  = 4  * 60 * 1000;  // check every 4 minutes
const REFRESH_THRESHOLD  = 5  * 60;          // refresh if < 5 min remaining (seconds)

export function TokenRefreshProvider({ children }: { children: React.ReactNode }) {
  const router      = useRouter();
  const queryClient = useQueryClient();

  const checkAndRefresh = useCallback(async () => {
    try {
      // Get token from cookie bridge
      const tokenRes = await fetch('/api/auth/token');
      if (!tokenRes.ok) return;
      const { accessToken } = await tokenRes.json() as { accessToken: string | null };

      if (!accessToken) return;

      // Decode exp claim from JWT payload
      const payload = JSON.parse(atob(accessToken.split('.')[1])) as { exp: number };
      const secondsUntilExpiry = payload.exp - Math.floor(Date.now() / 1000);

      // Token already expired — redirect to login
      if (secondsUntilExpiry <= 0) {
        console.log('[TokenRefresh] Token expired — redirecting to login');
        router.push('/login?reason=expired');
        return;
      }

      // Token expiring soon — proactive refresh
      if (secondsUntilExpiry < REFRESH_THRESHOLD) {
        console.log(`[TokenRefresh] Token expires in ${secondsUntilExpiry}s — refreshing`);
        const refreshRes = await fetch('/api/auth/refresh', { method: 'POST' });

        if (refreshRes.ok) {
          // Invalidate all queries so they refetch with new token
          queryClient.invalidateQueries();
          console.log('[TokenRefresh] Token refreshed successfully');
        } else {
          // Refresh failed — session truly expired
          console.log('[TokenRefresh] Refresh failed — redirecting to login');
          router.push('/login?reason=expired');
        }
      }
    } catch (err) {
      // Network error — don't redirect, user might just be offline temporarily
      console.warn('[TokenRefresh] Check failed:', err);
    }
  }, [router, queryClient]);

  useEffect(() => {
    // Check immediately on mount
    checkAndRefresh();

    // Check every 4 minutes while app is open
    const interval = setInterval(checkAndRefresh, CHECK_INTERVAL_MS);

    // Check when user returns to the tab after being away
    // This handles the "idle user" scenario
    const handleVisibilityChange = () => {
      if (document.visibilityState === 'visible') {
        console.log('[TokenRefresh] Tab became visible — checking token');
        checkAndRefresh();
      }
    };
    document.addEventListener('visibilitychange', handleVisibilityChange);

    return () => {
      clearInterval(interval);
      document.removeEventListener('visibilitychange', handleVisibilityChange);
    };
  }, [checkAndRefresh]);

  // This component renders nothing — it's purely a side-effect provider
  return <>{children}</>;
}

'use client';

import { useAuthQuery, useAuthMutation } from '@/modules/auth/hooks/useAuthQuery';

const USERS_QUERY = `
  query GetUsers {
    users {
      id
      email
      role
      status
    }
  }
`;

const UPDATE_USER_ROLE = `
  mutation UpdateUserRole($userId: String!, $role: String!) {
    updateUserRole(userId: $userId, role: $role) {
      id role
    }
  }
`;

interface User { id: string; email: string; role: string; status: string; }

export function UserList() {
  // Fetch users — auto-refreshes token if expired
  const { data, isLoading, error } = useAuthQuery<{ users: User[] }>(
    ['users'],    // cache key — invalidate this after mutations
    USERS_QUERY
  );

  // Mutation — invalidates users cache after success
  const updateRole = useAuthMutation<{ updateUserRole: User }, { userId: string; role: string }>(
    UPDATE_USER_ROLE,
    {
      invalidateKeys: [['users']],  // auto-refetch users after role update
      onSuccess: () => console.log('Role updated'),
    }
  );

  if (isLoading) return <div>Loading...</div>;
  if (error)     return <div>Error loading users</div>;

  return (
    <ul>
      {data?.users.map((u) => (
        <li key={u.id}>
          {u.email} — {u.role}
          <button
            onClick={() => updateRole.mutate({ userId: u.id, role: 'admin' })}
            disabled={updateRole.isPending}
          >
            Make Admin
          </button>
        </li>
      ))}
    </ul>
  );
}
